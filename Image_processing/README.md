# Image Processing Tool (NumPy)

This project implements basic image processing from scratch using NumPy.

## Features

* Grayscale conversion
* Image blurring (convolution)
* Edge detection (Sobel operator)
* Thresholding
* Visualization (subplot)
* Save output image

## Tech Used

* Python
* NumPy
* Matplotlib
* Tkinter

## How to Run

```bash
python image_processing.py
```

Select an image and view results.
